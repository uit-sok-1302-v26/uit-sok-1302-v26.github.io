# Kapittel 6: Sekvensielle spill – anvendelser

Dette kapittelet gir konkrete illustrasjoner av sekvensielle spill med perfekt informasjon og viser hvordan delspillperfekt likevekt brukes i praksis. Gjennom sentrale eksempler som ultimatumspillet, holdup-problemet, Stackelberg-modellen for duopol, stemmekjøp og konkurranse i kappløp, analyseres strategiske situasjoner der rekkefølgen på trekk, troverdige trusler og strategisk forpliktelse er avgjørende for utfallet. Kapittelet viser hvordan baklengs induksjon anvendes i spill med kort og lengre horisont, og hvordan likevektsutfall kan avvike betydelig fra både Nash-likevekter i simultane spill og fra intuitive eller samfunnsmessig ønskelige løsninger. Samlet gir kapittelet et helhetlig bilde av hvordan sekvensielle spill kan brukes til å forstå markedsmakt, forhandlinger, politiske prosesser og dynamisk konkurranse.

## Dette skal du kunne etter kapittelet

- Bruke baklengs induksjon til å analysere konkrete sekvensielle spill
- Finne delspillperfekte likevekter i spill med kort og lengre horisont
- Analysere ultimatumspillet og forklare hvorfor utfallet kan avvike fra eksperimentelle resultater
- Forstå holdup-problemet og hvorfor ineffektive utfall kan være stabile
- Analysere Stackelberg-konkurranse og forklare fordelen ved å være førstebeveger
- Sammenligne simultan og sekvensiell konkurranse i markedsmodeller
- Forstå hvordan troverdige og ikke-troverdige trusler påvirker strategiske valg
- Analysere dynamiske konkurransesituasjoner som stemmekjøp og kappløp
